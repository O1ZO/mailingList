# mailingList

- Nom de la base de donnée : mailingList


- 2 tables utilisées : "users" et "newsletters"

  - Champs de la table "users" :
    - id
    - login
    - name
    - firtsName
    - email
    - password
    - newsletterId
    - adminRights
    
   - Champs de la table : "newsletters" :
     - id
     - title
     - subject
     - content

- Mail : Gmail
